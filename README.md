ExploreNow 
Acesse o site online: https://explorenowprojeto.netlify.app

ExploreNow é um projeto de website para uma agência de viagens fictícia, desenvolvido com foco em desenvolvimento front-end, boas práticas de código e organização de projeto.


Descrição:

O projeto simula um site institucional de uma agência de viagens, com páginas informativas, seções interativas e estrutura preparada para evolução futura, incluindo autenticação de usuários e sistema de pagamentos.

Atualmente, o foco do desenvolvimento está no front-end.


Funcionalidades implementadas:

- Página Home com seção de destaque
- Página Sobre com apresentação institucional
- Diferenciais exibidos em cards
- Header fixo com navegação
- Efeitos de hover e transições
- Organização de CSS


Funcionalidades planejadas:

- Página de Destinos
  - Lista de destinos disponíveis
  - Informações detalhadas sobre cada destino
  - Serviços inclusos no pacote
    
- Página de Contato
  - Formulário para contato com a agência
- Página de Cadastro e Login
- Sistema de pagamento (fase futura)
- Integração com back-end (fase futura)



Tecnologias:

Atual
- HTML
- CSS
- JavaScript

Futuro:
- Python (back-end)
- Banco de dados
- Integração com sistema de pagamento

 Status do Projeto: em andamento
